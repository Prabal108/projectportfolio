About The Research

This project develops a multimodal physiological analysis framework to objectively evaluate human cognitive and emotional states within controlled Virtual Reality (VR) environments. By integrating EEG (brain activity), fNIRS (hemodynamic response), and PPG-derived PRV (autonomic response), the system captures complementary neural, vascular, and cardiac signals in real time. Two contrasting VR conditions—cognitive workload (chess task) and relaxation (guided meditation)—were designed to induce distinct mental states. Signal preprocessing, feature extraction (e.g., EEG band power, frontal asymmetry, PRV metrics), and machine learning models were applied to identify consistent physiological patterns across modalities. The results demonstrate clear multimodal separation between high cognitive load and relaxed states, validating the framework as a reliable tool for objective mental state assessment. This approach provides a data-driven foundation for applications in neuroadaptive systems, cognitive monitoring, and human-centered environment evaluation..

This framework can be extended to evaluate architectural design variables through measurable neurophysiological responses

Methodology

Input Systems

The research utilizes multi-modal input sources including eye tracking, pulse rate variability, EEG brainwave data, and pupillometry from the Meta Quest 3 system, combined with neural activity data from the Muse S Athena headset.

This comprehensive sensor fusion approach allows for a holistic understanding of user states during immersive VR experiences, capturing both conscious and subconscious responses to virtual stimuli.

Input Data Visualization

Output & Adaptive Systems

The system outputs objective assessments of cognitive and emotional states by analyzing multimodal physiological signals, including EEG, fNIRS, and PRV. It identifies clear distinctions between workload and relaxation conditions while providing interpretable insights into neural, hemodynamic, and autonomic responses, enabling reliable session-level evaluation

The framework can be extended to enable real-time adaptive VR environments based on detected user states.
